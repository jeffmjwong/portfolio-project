Projects & Apps

Traditional memory game is a simple yet popular game to play, suitable for players of all age. Players have to match all the cards to win the game. If a set of cards don't match after players revealed them, they will be hidden again, thus testing the player's memory. This version of memory game is built to be played on web browsers.

Technologies used: HTML5, CSS3, JavaScript, JQuery

Source code: https://github.com/jeffmjwong/memory-game-by-jeff

Try it here: https://jeffmjwong.github.io/memory-game-by-jeff/

This photo search engine is one of the projects that I built after exposure to AJAX and API technologies. Users can search for photos when they type in a keyword in the input field, and the app will make an AJAX call to request latest photos embedded with the keyword from Flickr via Flick's public API after the search button is clicked.

Technologies used: HTML5, CSS3, JavaScript, JQuery, AJAX, Fetch API

Source code: https://github.com/jeffmjwong/flickr-search-by-jeff

Try it here (have to turn off secure site to work): https://jeffmjwong.github.io/flickr-search-by-jeff/
